# **App Name**: GeoTracker

## Core Features:

- Location Tracking: Continuously track the user's longitude and latitude coordinates.
- Display Coordinates: Display the current longitude and latitude in a clear format.
- Map Integration: Show the location on an integrated map using the coordinates.
- Periodic Updates: Update location information on a regular interval (e.g., every 5 seconds).

## Style Guidelines:

- Primary color: Black (#000000) for a dark theme.
- Background color: Dark gray (#121212) for a comfortable dark backdrop.
- Accent color: Red (#FF0000) for highlights and important elements.
- Secondary color: White (#FFFFFF) for text and icons to ensure high contrast.
- Body and headline font: 'Inter' sans-serif for clear readability and modern appeal, optimized for dark mode.
- Use simple, recognizable icons with a white or light gray stroke for location-related functions, like a pin or target, to ensure visibility on a dark background.
- Keep the layout clean and straightforward, with coordinates and map prominently displayed, optimized for a dark theme.
- Subtle animations for location updates and map interactions, with consideration for reduced brightness and potential eye strain in a dark environment.
